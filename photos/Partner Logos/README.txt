HOW THE COLLAGE WORKS
  Logos are arranged biggest -> smallest by level of support (Ryan's ranking).
  The bigger the supporter, the larger the tile. To change a logo, drop a new
  file with the SAME filename below. To add/remove/re-rank a sponsor, send Ryan
  the change and we'll resize/reorder the tiles to match.
